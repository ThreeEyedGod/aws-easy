# s3-demo by Richard Cook

Haskell application with main executable only built using Stack

## Clone repository

```
git clone https://github.com/rcook/s3-demo.git
```

## Install compiler

```
stack setup
```

## Build

```
stack build --fast
```

## Run application

```
stack exec s3-demo-app
```

## Run tests

```
stack test
```
